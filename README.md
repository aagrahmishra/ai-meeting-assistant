# AI Meeting Assistant Platform

A real-time video communication platform that integrates AI agents for automated meeting summaries and action item generation.

## Tech Stack
- React, TypeScript, Next.js
- Node.js, Stream SDK
- PostgreSQL (Neon), Drizzle ORM
- JWT Authentication

## Features
- Real-time video calls with AI agent integration
- Custom UI components for AI agent indicators
- Automated meeting transcript generation
- Session management for concurrent users
- Sub-150ms query response times
